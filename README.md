# Air-Gapped Optical Covert Channel

## Overview
This project explores optical covert communication from air-gapped systems using screen brightness modulation and smartphone camera capture.

## Motivation
Air-gapped systems are considered secure because they are isolated from networks. This project investigates optical side-channel leakage using commonly available devices.

## Methodology
- Screen brightness modulation
- Binary data encoding
- Smartphone camera recording
- Frame extraction and brightness analysis
- Data reconstruction

## Setup
- Computer screen
- Smartphone camera
- Python for analysis
- Controlled indoor environment

## Results
- Successful data transmission over short distance
- Works under indoor lighting conditions
- Limited bandwidth but reliable transmission

## Applications
- Air-gap security research
- Side-channel analysis
- Hardware security research

## Future Work
- Improve data rate
- Use LED-based transmission
- Multi-channel optical transmission
